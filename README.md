# Odoo 17 Real Estate Module
Real Estate Management System built with Odoo 17
## Custom Modules
- **todo_module**: Task management system with activity tracking
- **school**: Educational management module
- **minia**: Real estate management
## Features
- Task creation and tracking
- Due date management
- Time tracking and estimation
- Activity logging and chatter
- Status workflow (New → In Progress → Completed → Closed)
## Installation
1. Copy custom_addons to your Odoo addons directory
2. Go to Apps → Update Apps List
3. Search for and install the desired module
